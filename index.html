Secure Close Browser

A simple way to clear common browser privacy artifacts (history, cookies, cache, sessions) after you close your browser.

Important safety note
  • This is intended for your own device for security/privacy hygiene.
  • It avoids deleting bookmarks and saved passwords, but it may sign you out of websites.
  • Always close the browser before cleaning, and test with --dry-run first.
Best built-in option (recommended)

Firefox supports “Clear history when Firefox closes” in Settings → Privacy & Security. Chromium browsers are more limited; use the launcher script below for reliable clearing.

Quick start

  1. Install Python 3 (if you don’t have it).
  2. Save the script as secure_close_browser.py.
  3. Run one of these:
# Chrome
python secure_close_browser.py --browser chrome

# Edge
python secure_close_browser.py --browser edge

# Brave
python secure_close_browser.py --browser brave

# Firefox
python secure_close_browser.py --browser firefox

# Test safely (no deletion)
python secure_close_browser.py --browser chrome --dry-run
